SET09103 CourseWork 2 Project 

Below is a project made for Advanced Web Technologies. The project is a Pyhton Flask Micro Framework 

It was developed in a virtual envirnonment with the napier server. Below are the additional libraries/modules used to make this project work:


flask-wtf - to generate login, register forms etc
flask-sqlalchemy - for the database
flask-migrate - migration for the databse
flask-login - to allow users to login
flask-moment - to display last seen time on user profile
PIL - to allow pictures in post

in order to run this project you will need the following commands within the server: 

export FLASK_APP=blog.py
export FLASK_ENV=development
python -m flask run --host=0.0.0.0 --port 9158
